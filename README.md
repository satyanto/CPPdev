learning c plus plus. 



to compile on MacOS:
```g++ -o [name of executable] [cpp file name].cpp```

apparently to compile w/ threading (only c++11 and above):
```g++ -std=c++11 -pthread -o [name of executable] [cpp file name].cpp```

to run on MacOS terminal:
```./[name of executable]```
